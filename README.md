# Preference-Based Place Recommendation System

## 프로젝트 개요

사용자의 장소 선호도 데이터를 기반으로 유사한 성향의 사용자를 군집화하고, 각 군집의 선호 패턴을 분석하여 맞춤형 장소를 추천하는 추천 시스템 프로젝트입니다.

사용자별 선호도 설문 데이터를 활용하여 K-Means 군집화를 수행하였으며, 군집별 선호 장소를 분석하여 사용자에게 개인화된 장소 추천이 가능하도록 구현하였습니다.

## 데이터 구성

사용자 설문 데이터를 활용하여 장소별 선호도를 수집하였습니다.

데이터 예시

* 사용자 ID
* 장소별 선호도 점수
* 카페 선호도
* 기타 장소 선호도

---

## 프로젝트 수행 과정

### 1. 데이터 전처리

* 사용자 ID 분리
* 분석용 데이터셋 구성
* 결측치 및 데이터 구조 점검

### 2. 사용자 군집화

K-Means 알고리즘을 활용하여 사용자 그룹을 군집화하였습니다.

* K-Means Clustering
* 사용자 성향 그룹 분류
* 군집별 특성 분석

### 3. 군집별 선호 장소 분석

각 군집의 평균 선호도를 계산하여 가장 높은 선호도를 보이는 장소를 도출하였습니다.

* 군집별 평균 선호도 계산
* 최다 선호 장소 분석
* 추천 후보 추출

### 4. 신규 사용자 추천

신규 사용자의 설문 데이터를 기존 데이터와 함께 학습하여 군집을 판별하고, 해당 군집에서 가장 선호도가 높은 장소를 추천하도록 구현하였습니다.

## 배운 점

본 프로젝트를 통해 추천 시스템에서 단순히 사용자별 개별 추천을 수행하는 것보다, 유사한 성향을 가진 사용자를 군집화하여 추천하는 방법의 효과를 경험할 수 있었습니다. 또한 K-Means 군집화와 사용자 행동 데이터 분석을 통해 데이터 기반 개인화 추천 시스템의 기본 원리를 이해할 수 있었습니다.
